# Remal Image: Java 21 Omni Runner

Ultimate Java-21 runner image for Spring Boot

Details:
* PostgreSQL database server
* Java fonts

Appendix 1) Install Java fonts

Install fonts all in one:
~~~
$ apk add /
    font-terminus /
    font-inconsolata /
    font-dejavu /
    font-noto /
    font-noto-cjk /
    font-awesome /
    font-noto-extra /
    font-terminus
~~~
